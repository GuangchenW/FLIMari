# FLIMari Documentation

Welcome to the documentation of FLIMari [GitHub repository](https://github.com/GuangchenW/FLIMari).

FLIMari is an open-source [Napari](https://napari.org) plugin for machine learning analysis of fluorescence lifetime images using the [Phasor approach](https://www.phasorpy.org/docs/stable/phasor_approach/).

!!!warning
    This documentation is currently under construction. Contents may be missing or subject to changes.

The [Tutorials](user-guide/tutorials.md) showcase how to use the FLIMari plugin.
