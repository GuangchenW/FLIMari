# FLIMari Documentation

Welcome to the documentation of FLIMari.

FLIMari is an open-source [Napari](https://napari.org) plugin for machine learning analysis of fluorescence lifetime images using the [Phasor approach](https://www.phasorpy.org/docs/stable/phasor_approach/).

[FLIMari GitHub repository](https://github.com/GuangchenW/FLIMari)

!!!warning
    This documentation is currently under construction. Contents may be missing or subject to changes.

## How to Install

Follow the [napari tutorial](https://napari.org/dev/tutorials/fundamentals/installation.html) to set up the python environment, then run the following command in the same environment:

`pip install "git+https://github.com/GuangchenW/FLIMari.git"`
